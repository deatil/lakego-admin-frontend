## lakego-admin 后台管理系统前端项目

### 项目介绍

*  项目基于 `vue-vben-admin` 项目开发
*  使用 `Vite`, `Vue3`, `Ant-Design-Vue` 开发
*  本项目为 `lakego-admin` 的前端项目

###  安装使用

- 获取项目代码

```bash
git clone https://github.com/deatil/lakego-admin-frontend.git
```

- 安装依赖

```bash
cd lakego-admin-frontend

yarn install

```

- 运行

```bash
yarn serve
```

- 打包

```bash
yarn build
```
